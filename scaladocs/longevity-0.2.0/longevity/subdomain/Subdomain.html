Subdomain

Related Doc: package subdomain

case class Subdomain(name: String, entityTypePool: EntityTypePool, shorthandPool: ShorthandPool = ShorthandPool()) extends Product with Serializable

a specification of a subdomain of a project's domain. contains a pool of all the entity types in the subdomain, as well as all the shorthands used by the entities.

name

the name of the subdomain

entityTypePool

a complete set of the entity types within the subdomain

shorthandPool

a complete set of the shorthands used by the bounded context. defaults to empty
